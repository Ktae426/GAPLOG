# 📘 GAPLOG  
> 경력 단절자를 위한 단기 체험 제공 어플리케이션

<br>

## 🚀 프로젝트 소개
**GAPLOG**는 육아, 건강 등의 이유로 경력이 단절된 사람들에게 다시 사회로 복귀할 수 있다는 자신감 회복과 실제 업무 감각을 되찾을 수 있도록 단기 체험 기회를 제공하는 서비스 입니다.

<br>

## 🧑‍💻 주요 기능 
- 🔐 로그인 / 회원가입
- 📊 단기 체험 검색
- 📅 리턴맵
- 👥 커뮤니티
- 📝 내 정보

<br>

## 🛠 기술 스택
- UI/UX: Flutter, Dart
- 데이터베이스: Firebase
